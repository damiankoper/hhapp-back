Set on: 
* Docker@19.03.1
* DockerCompose@1.24.1

## Run for development
```
npm run docker:dev
```
### Test watcher
After launching `dev` docker services:
```
npm run docker:dev:test
```

## Run for production
```
npm run docker:prod
```
